## Built With
* Pure PHP
* MySQl
* AngularJS
* BootStrap 4
## Config file
The config file is: config.php!
In this file you set:
* Apis urls
* DataBase connection config
* Sleep time between resending unsent messages (in seconds)
## dk-sms.sql
This file is for initializing your database.
